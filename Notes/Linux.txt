pwd - print working directory
mkdir <fileName> - make directory (Create new folder)
rmdir <fileName> - remove directory
ls - list contents of a directory
cd  <toDirectoryName> - change directory
touch <fileName> - Creates empty file 
cat 
The cat command is a multi-purpose utility in the Linux system. It can be used to create a file, display content of the file, copy the content of one file to another file, and more

mv - move one file from one directory to another directory
cp - make copy within the same directory
rename - renames the file


head - displays the first 10 lines of the file
tail- displays the last 10 lines of the file

tac - reverse of cat. Displays all the content of a file in reverse order
more - similar to cat command, but can display large content file
		Enter key - scroll down
		Space bar - move to next page 
		b key     - move to previous page
		/ key 	  - search the string
		
less - similar to more but less features


su - enables admin access for users
id - displays id and grp id of the users


> redirection operator
